**Компоненты:**
1. **Backend приложение** - простой HTTP-сервер на Python, слушающий порт 8080
2. **Nginx** - reverse proxy, принимающий запросы на порт 80 и проксирующий их на backend
3. **Docker Compose** - оркестрация контейнеров и сетевого взаимодействия

## Технологии

- Docker 20.10+
- Docker Compose 5.0.2
- Python 3.11
- Nginx 1.24
- Alpine Linux (для минимальных образов)

## Быстрый запуск

### Требования
- Установленный Docker
- Установленный Docker Compose
- Порты 80 и 8080 свободны на хосте

### Запуск приложения

1. **Клонируйте репозиторий:**
   bash
   git clone <repository-url>
   cd effective-mobile-devops-test

2. **Создание контейнеров и запуск служб**
   docker compose up -d

3. **Проверка работы проекта**
   curl http://localhost

### Детальная схема потоков данных:
1. **Пользователь** → `http://localhost:80` (запрос с хоста)
2. **Docker** → перенаправление в контейнер nginx (проброс порта 80:80)
3. **Nginx** → получает запрос, проверяет конфигурацию
4. **Nginx** → резолвит DNS имя `backend` через Docker сеть
5. **Nginx** → проксирует запрос на `backend:8080`
6. **Backend** → Python приложение обрабатывает GET запрос
7. **Backend** → возвращает "Hello from Effective Mobile!"
8. **Nginx** → передаёт ответ пользователю
